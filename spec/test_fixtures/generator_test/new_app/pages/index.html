CSS

Dump all your CSS/SCSS files in the 'css' folder. Call them all with a single call to 'application.css'

JS

Dump all your JS/Coffee files in the 'js' folder. Call them all with a single call to 'application.js'

Layouts

Put your layout in the 'layouts' folder. Call 'render_page' anywhere in it & Rush will bring the correct page.

Partials

No more repeating HTML ever. Put it in a partial and call it from anywhere with render_partial.
